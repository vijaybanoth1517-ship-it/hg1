Welcome to the community portal for Hackers Gurukul students!

1. Database Setup

Create a MySQL database named hg_community and update the database credentials in config/database.php

2. Web Server

Make sure you're running this on a PHP-enabled web server (Apache/Nginx with PHP 7.4+ and MySQL)

3. Create Admin Account

Run the database migrations by accessing any PHP file - tables will be created automatically. Then manually insert an admin user into the database.

4. File Permissions

Ensure the uploads/ directory is writable by your web server

Note: This is a development setup. For production, ensure proper security measures, HTTPS, and environment-specific configurations.

Once setup is complete, navigate to login.php to start using HG Community!
